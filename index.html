⚙️ Settings & Integrations
🔐 Verification Services
Booking Worker URL (Firebase Sync)
Cloudflare Worker URL for Firebase sync and booking actions. Deploy prestige-black-booking-worker.js to get this URL.
ID Verification Worker URL
Cloudflare Worker URL for Stripe Identity verification. Deploy prestige-black-verification-worker.js to get this URL.
Messaging Worker URL
Cloudflare Worker URL for sending emails (review requests, etc.). Deploy prestige-black-messaging-worker.js to get this URL.
Owner Notification Email
Receives new booking alerts and ID verification emails. Leave blank to disable.
Date Lock Payment Link ($100)
Stripe payment link for the $100 date lock fee. Included in booking confirmation emails so customers can pay immediately.
Balance Payment Link (Stripe)
Stripe payment link for remaining balance. Shown in the customer portal so clients can pay their remaining balance online.
Google Review URL
Sent to customers after trip is marked complete. Find yours at business.google.com.
OpenAI API Key
Used for AI insurance card scanning (~$0.01/scan with GPT-4o). Get one at platform.openai.com.
✍️ Owner Signature
Signature Image
No signature uploaded — contracts will leave the line blank
PNG or JPG with transparent/white background works best. Stored locally in this browser only.
Privacy note: API keys are stored only in this browser's localStorage. Your signature is synced to Firebase so it appears on all your devices.
🔒 PIN Lock
Require a PIN when opening the dashboard in a new browser session. Does not encrypt data — UI-level access control only.
👥 Staff Access
Add staff PINs with limited roles. Check-in Only: can manage bookings but can't see revenue or finances. Read Only: view bookings only.
🗃 Recently Deleted
Bookings deleted in the last 30 days can be restored. Older deletions are permanent.
📋 Audit Log
Full chronological history of every booking change — who made it, when, and what changed.
🏍 Contract Templates
Edit the rental agreement and Slingshot liability waiver templates used in contracts.
⛔ Blacklisted Guests
Blacklisted guests are flagged on all bookings. To blacklist a guest, open any booking and tap ⛔ Blacklist.
Danger Zone
Permanently deletes all bookings, expenses, income, and maintenance records from this device and Firebase. This cannot be undone.
📬 Sent Notifications
⚠ Unsaved changes — click Save Settings to apply
🗃 Recently Deleted Bookings
Deleted within the last 30 days. Restoring a booking makes it active again.
